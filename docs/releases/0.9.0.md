## Release boundary

| Classification | Included in 0.9.0 |
| --- | --- |
| Stable/public | Source declarations, proposal/review/approval workflow, append-only audit, pending isolation, public rebuildable `CompiledWorkspace`, and v1/v2 navigation and read contracts |
